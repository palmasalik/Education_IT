Программа курса

Изучение среды разработки Visual studio code, графического редактора Figma и основы создания сайтов с помощью HTML и CSS.

Полное погружение в самый гибкий язык программирования JavaScript для создания современных и по-настоящему интерактивных сайтов.

Знакомство с прогрессивным фреймворком Vue.js - для создания SPA приложений, Vuex - с паттерном управления состоянием и библиотекой для приложений на Vue.js, Vue-router для создания MPA приложений.

Грамотное структурирование кода - БЭМ, надежное хранение проектов и загрузка портфолио на облачный сервис GITHUB с помощью системы контроля версий Git.

Два популярных сборщика GULP, WEBPACK для автомотизаци и оптимизации кода, так же дополнительные инструменты SASS, PUG для упрощения верстки и разработки сайта.

PHP один из самых популярных языков программирования в backend - серверная часть сайта, так же знакомство с самой популярной CMS системой WordPress.
